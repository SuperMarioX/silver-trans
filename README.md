# silver-trans
基于Netty实现的文件传输组件.

|        命令        |                 | 
| ----------------- |-------------------:|
| connect ip:port   | 连接到主机|
| disconnect        | 断开连接|
| exit              | 退出程序|
| pwd               | 显示当前路径|
| cd *              | 进入目录|
| ls                | 显示当前目录文件列表|
| get               | 获取文件|
